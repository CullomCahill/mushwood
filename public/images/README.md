# Images Folder

Place all your product images, hero images, and other photos in this folder.

## Image Paths

- **Hero image (home page)**: Update `server.js` line 50 to use `/images/your-hero-image.jpg`
- **About page image**: Update `server.js` line 113 to use `/images/your-about-image.jpg`
- **Product images**: Update paths in `lib/products.js` to use `/images/your-product-image.jpg`

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Naming Tips

- Use lowercase letters and hyphens: `mushroom-shelf.jpg`
- Keep filenames descriptive but short
- Use consistent naming: `product-name.jpg` or `product-1.jpg`

## Example Structure

```
public/images/
  ├── hero.jpg
  ├── about.jpg
  ├── mushroom-shelf.jpg
  ├── forest-stool.jpg
  └── ...
```
